# Marketplace Listing

## Name

Frontend Polish

## Short Description

Make generated or rough frontend UI feel intentional, responsive, and product-grade while preserving business logic.

## Long Description

Frontend Polish gives Codex a senior frontend design-engineering workflow for improving UI quality without rewriting the app underneath. It helps with visual hierarchy, spacing rhythm, typography, responsive behavior, interaction states, surface treatment, and maintainable implementation.

Use it after a generated frontend works but still feels templated, repetitive, oversized, or unfinished. It is useful for React, Vue, Tailwind CSS, and plain HTML/CSS projects, including landing pages, dashboards, auth flows, product detail pages, ecommerce screens, and individual UI components.

The skill is intentionally restrained: it favors clarity, structure, density, state quality, and code maintainability over decorative effects. It helps Codex avoid common AI UI smells such as card soup, generic gradients, weak focus states, awkward mobile stacking, and visual drama that hides unclear hierarchy.
